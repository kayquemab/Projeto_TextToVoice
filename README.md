# 📌 TextToVoice

Aplicação web que converte texto em áudio utilizando a Web Speech API.

## 🚀 Tecnologias

* HTML5
* CSS3
* JavaScript
* Web Speech API

## ✨ Funcionalidades

* Conversão de texto em voz
* Seleção de vozes disponíveis

## ⚙️ Como executar

Baixar ou clonar o repositório e abrir o arquivo `index.html` no navegador

## 📌 Observações

* Projeto estático (não requer instalação)
* As vozes podem variar conforme o navegador
